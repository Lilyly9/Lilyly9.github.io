## HTML
### HTML简介
1. 全称：Hyper Text Markup Language（超文本标记语言） 
2. 作用：专门用来搭建网页的结构（骨架），比如文字、图片、按钮、表单、布局等。
### 了解基本骨架
```html
<!-- 文档声明：告诉浏览器这是HTML5文档 -->
<!DOCTYPE html>
<!-- 网页语言是中文（简体） -->
<html lang="zh-CN">
  <!-- 头部：给浏览器看的配置，不显示在页面上 -->
  <head>
    <meta charset="UTF-8">
    <!-- 设置网页字符编码为 UTF-8，保证中文不乱码 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- 网页适配手机，宽度跟随设备宽度，初始缩放比例 1:1 -->
    <title>我的第一个网页</title>
    <!-- 显示在浏览器标签栏上-->
  </head>

  <!-- 身体：页面所有可见内容都写在这里 -->
  <body>
    网页的文字、图片、按钮都在这里！
  </body>
</html>
```
### 标签
1. 基础文本标签
<span id="基础文本标签"></span>
    ```html
    <h1>一级标签</h1>
    <p>一段文字</p>
    <div>容器</div>
    <strong>加粗</strong>
    <em>斜体</em>
    <del>删除线</del>
    <ins>下划线</ins>
    <br>    <!-- 换行 -->
    <hr>    <!-- 水平分割线 -->
    ```
2. 超链接标签
   <span id="超链接标签"></span>
```html
<!-- 基本用法 -->
<a href="https://www.baidu.com">百度一下</a>
<a href="https://www.baidu.com" target="_blank">新页面打开</a>
```
href="地址"跳转的目标链接 &emsp;&emsp; target="_blank"  在新标签页打开链接

3. 图片标签
<span id="图片标签"></span>
```html
<img
        src="https://www.baidu.com/logo.png"
        alt="百度Logo"
        width="300" height="200">
>
```
src="图片地址" &emsp; &emsp;alt = alternative（替代文字）图片加载失败时，显示这段文字

4. 列表标签
    1. 无序列表 ul + li
       ```html
        <ul>
          <li>列表项1</li>
          <li>列表项2</li>
          <li>列表项3</li>
        </ul>
        ```
   2. 有序列表 ol + li
        ```html
        <ol>
          <li>第一项</li>
          <li>第二项</li>
        </ol>
        ```
   3. 自定义列表 dl + dt +dd
        ```html
        <dl>
          <dt>标题</dt>
          <dd>描述内容</dd>
        </dl>
        ```
   
5. 表格标签
    1. 基础表格 行tr  格子th/td（Table Header/Table Data）
    ```html
    <table border="1" width="400"> <!-- border=1 加边框，宽度width -->
      <tr>    <!-- 第一行：表头 -->
        <th>姓名</th>
        <th>年龄</th>
        <th>城市</th>
      </tr>
    
      <tr>     <!-- 第二行 -->
        <td>张三</td>
        <td>20</td>
        <td>北京</td>
      </tr>
    
      <tr>    <!-- 第三行 -->
        <td>李四</td>
        <td>22</td>
        <td>上海</td>
      </tr>
    </table>
    ```
   2. 合并单元格 rowspan/colspan="x"
   ```html
    <table border="1">
      <tr>
        <th colspan="2">学生信息</th> <!-- 合并2列 -->
      </tr>
      <tr>
        <td rowspan="2">张三</td> <!-- 合并2行 -->
        <td>20岁</td>
      </tr>
      <tr>
        <!-- 这里原本应该有<td>，但被合并了，所以删掉！ -->
        <td>男</td>
      </tr>
    </table>
    ```
6. 表单标签
    ```html
    <!-- 文本框 -->
    <input type="text" placeholder="请输入用户名">
    <!-- 密码框 -->
    <input type="password" placeholder="请输入密码">
    <!-- 单选框 -->
    <input type="radio" name="gender"> 男
   <input type="radio" name="gender"> 女
    <!-- 复选框 -->
    <input type="checkbox"> 看书
    <!-- 提交按钮 -->
    <input type="submit" value="登录">
    <!-- 普通按钮 -->
    <input type="button" value="点击">
    ```

7. 下拉菜单 <select> + <option>
    ```html
    <select>
      <option>北京</option>
      <option selected>上海</option>     <!-- 默认选中 -->
      <option>广州</option>
      <option>深圳</option>
    </select>
    ```
8. 转义字符
    ```html
    空格	&nbsp;
    <	&lt;
    >	&gt;
    &	&amp;
    人民币	&yen;
    版权	&copy;
    ```

## CSS

### css 简介

CSS 是层叠样式表(Cascading Style Sheets)的简称。
<br>CSS 主要用于设置HTML页面中的文本内容、图片外形以及版面的布局和外观显示样式。

### 选择器

1. **标签选择器**
    ```css 
   h3 {
        color: pink;
        font-size: 20px;
   }
   ```
2. **id选择器**
   ```css
   #d {
      color: darkblue;   
   }
   ```
3. **类选择器**
    1. 基本写法
       ```css
       .c {
          color: pink;
       }
       ```
       ```html
       <div class="c"> 粉色 </div>
       ```
    2. 多类名选择器
       <br>无空格：同时选中 <u>c1、c2</u>，优先级更高，会覆盖单个类名选择器
       ```html
       <div class="c1 c2"> 多类名选择器 </div>  /* 有空格 */
       ```
       ```css
       .c1.c2{   /* 无空格 */
         color: blue;
       }
       ```
    3. 后代选择器
       <br>选中 c1 里的子元素 <u>c2</u>
       ```css
       .c1 .c2{
         color: yellow;
       }
       ```
4. 通配符选择器
   <br>给所有的元素使用样式
    ```css
   * {
      margin: 0;
      padding: 0;
   }
    ```

### 字体属性
    
    body{font: italic bold 20px/1.5 "Microsoft YaHei";} 
    /* 字体格式 字体粗细 字体大小/一行文字高度 “字体” */

| font-style           |font-weight|font-size| line-height |font-family|
|----------------------|-----|-----|-------------|-----|
| **italic** 斜体/normal |字体粗细 normal(400)/bold(700)|字体大小 20px| 一行文字所占的高度   |字体|

### 文本属性

1. 文字颜色
   |预定义的颜色值|十六进制|RGB代码|
   |---------|--------|--------|
   |blue/red…|#ff0000|rgb{255,0,0}/rgb{100%,0%,0%}|
2. 水平对齐
    ```css
    p{
       text-align: left;
       text-align: center;
       text-align: right;
    }
   ```
3. 文本装饰 **text-decoration**
    |默认|下划线|上划线|删除线|
    |------|------|-----|-----|
    |none|underline|overline|line-through|
4. 文本缩进
    <br>**text-indent**： 2em;
   <br>**em** 是一个相对单位，就是当前元素（font-size) 1 个文字的大小, 如果当前元素没有设置大小，则会按照父元素的 1 个文字大小。

5. 行间距 **line-height**
   line-height: 1.5;
   line-height: 30px;
6. 字间距 **letter-spacing**
   letter-spacing: 2px;

### 样式表
1. 行内样式表
    <br>写在标签里
    ```html
    <div style="color:red; font-size:20px;">我是红色文字</div>
    ```
2. 内部样式表
   <br>写在 head 里
    ```html
    <head>
      <style>
        div {
          color: blue;
          font-size: 20px;
        }
      </style>
    </head>
    ```   
3. 外部样式表
    新建 style.css，在HTML中引入，放在head
    ```html
    <link rel="stylesheet" href="style.css">
   /* rell=引入的是stylesheet样式表;href=引入的css文件路径*/
    ```
