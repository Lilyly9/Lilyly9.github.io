## Node.js
1. 是什么？
   <br>是一个能让JavaScript跑在电脑本地（服务器/Windows电脑上）的运行环境。
   <br>浏览器里的JS只能在网页运行，Node.js 让 JS 变成了能直接操作电脑文件、装软件、跑脚本的 **后台程序**
   <br>**npm** 是 Node.js 自带的软件商店/包管理器，要装的 claude-code 就是发布在 npm 上的一个命令行工具，必须靠 Node.js + npm 来安装。
